# 钡铼技术资料库

欢迎访问钡铼技术开发者中心。
本站汇集全系列公开技术资料：设备快速上手部署指南、Node-RED 可视化数据流开发实战教程、Modbus/OPC UA/CAN 等工业通信协议说明、整机硬件规格书、接线手册、行业应用方案。

## 特色

- Node-RED 低代码可视化流程开发
- Modbus RTU/TCP 工业总线通信
- MQTT 协议云端对接上云
- OPC UA 跨平台工业数据交互集成
- CAN 总线现场设备通信
- 嵌入式 Linux 边缘计算运行环境

## 应用

- 工业物联网边缘网关
- 远程设备监控与管控
- 边缘端实时数据采集
- 多协议云端对接
