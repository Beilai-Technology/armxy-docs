# ARMxy Developer Center

Welcome to the official documentation center for ARMxy industrial edge gateways.

ARMxy is an industrial edge computing platform designed for industrial IoT applications, supporting multiple industrial protocols and edge applications.

## Features

- Node-RED workflow development
- Modbus RTU / TCP communication
- MQTT cloud connectivity
- OPC UA industrial data integration
- CAN bus communication
- Linux edge computing environment
